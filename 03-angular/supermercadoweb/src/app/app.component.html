Un texto de prueba es un pasaje de las Escrituras presentado como prueba de una doctrina, creencia o principio teológico. Prooftexting (a veces "prueba de texto" o "prueba de texto de texto") es la práctica de usar citas de un documento, ya sea con el propósito de exégesis o para establecer una proposición en eisegesis (introduciendo las propias presuposiciones, agendas o prejuicios).
